HTML 클래스와 아이디 속성을 알아보자

모든 HTML요소는 class와 id속성을 가지며 두 속성은 선택자로 이용된다.

복수의 선택자 class

클래스는 모든 html요소에 하나 이상의 이름을 가질 수 있다. 이때 클래스는 중복 가능하다.

  1. 규칙: class="이름"
  2. 규칙: class="이름1 이름2 이름3 이름4 ..."
  3. 규칙: .className{ color:red; }
  4. 규칙: document.getElementsByClassName("className")[n]
  5. [] : 배열의 자료형을 의미.
    예) ["토마토", 30, 0.12,"런던"]길이가 4인 배열
  6. class는 복수의자료를 처리할 때 사용한다.

  1. 규칙: id="idName"(o)

    id="3name"(권장x)

    document내에서 id의 유일성을 높여준다.
  3. 규칙: #idName{ color:blue; }
  4. 규칙: document.getElementsById("idName")
