html 제출양식 form

html 양식은 사용자 입력을 수정하기 위해 사용된다. 예)설문조사, 회원가입, 로그인,...

양식(form)에 입력한 복수의 데이터(파라미터)는 처리를 위해 서버로 전송(post방식)된다.

form 태그

form 요소는 사용자 입력을 위한 html 양식을 만드는데 사용되며 입력양식을 1개 이상 포함하는 컨테이너다.
이때 입력 양식에는 text field, checkboxs, radio, submit... 등이 있다.
form과 입력요소는 모두 name을 가지고 이 name은 파라미터 이름으로 서버에 전송된다.
예)url? name=value & name2=value2
document.forms["formName"]["inputName"]로 element를 참조할 수 있기 때문에
formName은 아이디처럼 유일한 값이다. (<=> document.forms.formName.inputName.value)

form 태그 속성

  • action : 양식을 제출하는 url위치
  • method : 양식 데이터를 보낼 때 통신(HTTP)할 방법
    • method='GET'(default) : url에 파라미터를 이름/값 쌍을 추가하는 방법으로 파라미터가 안전하지 않다.(url이 드러남..) 예) url?name=value&name=value(안전하지 않다.)
    • method='POST' : 양식 파라미터를 서버에 보낼때 HTTP포스트 트랜잭션에 추가해서 보낸다.(파라미터가 안전하다)
  • name : html 문서에서 유일한 이름이어야 한다.
  • autocomplete : 입력 양식의 자동완성(autocomplete="on 또는 off")
  • novalidate : 양식 제출 시 유효성 검사를 하지 않도록 지정
  • target : 양식 제출 시 받은 응답을 표시할 창의 위치(양식을 서버에 제출하면 양식을 처리후 이동할 페이지를 반환)
    예)양식을 서버에 제출하면 처리한 상태에 따라 이동할(응답) 페이지가 다르다. ()로그인 실패->로그인, 로그인 성공->홈)
    • _self(default) : 동일한 프레임(창)에 표시
    • _blank : 새창이나 프레임에 표시
    • _parent : 상위 프레임에 표시
    • _top : 창 전체에 표시
  • enctype : 양식 데이터를 서버에 제출할 때 인코딩 방식을 지정(GET-url 인코딩이 고정, POST만 변경 가능)

입력 양식 요소

input은 입력양식 요소의 대표이며 여러 타입의 값을 전달할 수 있다.
모든 입력 양식 요소는 name과 value를 가지며 양식을 제출(submit)할 때 url 또는 포스트트렌잭션에 추가된다.

  • input : 다양한 입력 타입이 존재하는 입력요소
  • textarea : 여러줄의 텍스트 영역 제공
  • select : 드롭다운방식(optgroup:목록을 그룹으로 묶는 것)으로 목록을 정리가능
  • datalist : 입력하기 전에 미리 정의된 옵션 목록을 지정
  • button : 클릭이 가능한 버튼
  • label : 입력요소에 라벨을 정의(입력요소.id==label.for 또는 label>입력요소) 또는

input태그가 가지는 여러 타입의 입력요소

  • type="text" : 한줄 텍스트를 정의
  • type="search" : 검색 문자열을 입력하기 위한 필드
  • type="email" : 이메일을 입력하는 필드(제출시 자동 유효성검사)
  • type="url" : url을 입력하는 필드(제출시 자동 유효성검사)
  • type="password" : 입력한 문자를 암호화시킨다.
  • type="number" : 숫자를 입력하는 필드
  • type="button" : 클릭이 가능한 버튼(동작에 대하여서는 자바스크립트로 정의)
  • type="submit" : action에 정의된 페이지(서버)로 양식을 제출
  • type="reset" : value를 재설정 하는 버튼 (reset()실행)
  • type="range" : 범위를 제어하는 컨트롤 ui 제공
  • type="checkbox" : 선택한 파라미터만 전달 (체크 한것에 한해 전달하는 것일 뿐)
  • type="radio" : 같은 이름의 파라미터가 복수의 value 중 하나만 선택해서 전달
  • type="color" : 색상 선택기
  • type="datetime-local" : 데이트픽커 UI제공 value="2021-08-21 T17:58"
  • type="date" : value="2021-08-21"
  • type="month" : value="2021-08"
  • type="week" : value="2021-w34"
  • type="time" : value="17:58"
  • type="file" : 파일 업로드(탐색기 제공)
  • type="hidden" : 숨겨진 파라미터
