html의 javascript

자바스크립트는 document object가 생성된 후 실행되며 화면에 동적 기능을 넣는다.
요소의 속성으로 on으로 시작하는 이벤트 속성은 특정 이벤트 발생시 동작한다.
html은 웹 컨텐츠의 구조와 의미를 정의하고 (구조적 언어)
css는 구성된 html을 꾸며주며
javascript는 브라우저에서 유저의 이벤트를 받아 html(dom)과 bom, css를 수정한다.

자바스크립트가 할 수 있는 일의 목록

  1. 모든 html요소(dom)을 변경할 수 있다.
  2. 모든 html요소의 속성을 변경할 수 있다.
  3. 모든 html요소의 스타일을 변경할 수 있다.
  4. 기존 요소와 속성을 제거할 수 있다.
  5. 새로운 요소와 속성을 추가할 수 있다.
  6. 모든 이벤트에 반응할 수 있다.
  7. 새로운 이벤트를 생성할 수 있다.

DOM Tree thml문서 객체모델

HTML DOM은 html을 위한 표준 객체 모델로 브라우저의 프로그래밍 인터페이스다.

  • HTML요소를 object객체로 생성 후 DOM TREE로 구성
  • 모든 html속성을 property로 구성함
  • 모든 html object를 엑세스 하는 방법을 제공
  • 모든 html요소의 이벤트 속성을 제공함

Java Script HTML DOM Document

웹페이지를 조작하려면 dom요소를(속성, 스타일) 수정, 삭제, 추가해야 하는데
이를 위해서 우선 document를 엑세스(참조) 부터 해야한다.

Finding HTML Elements (Document 엑세스하기)

요소를 수정하기 위해서는 요소 하나만 엑세스해야한다.
만약 HTML Collection을 엑세스하면 배열에서 반드시 특정 element를 선택해야한다.(HTMLCollection[i])
css는 복수의 요소를 모두 수정할 수 있다. 예) .className{}, div{},...
HTMLCollection = [element, element, element, ...] : 배열
getElementById(id) => element(단수)

  • document.getElementById("id") : id로 하나의 element를 엑세스 한다.
  • document.getElementsByClassName("className") : class Name으로 복수의 elements(HTML Collection)를 엑세스한다.
  • document.getElementsBytagName("tagName") : tag Name으로 복수의 elements(HTML Collection)를 엑세스한다.
  • document.querySelector("cssSelector") : . 또는 #과 같은 css선택자로 엑세스하는 하나의 element를 택
  • document.querySelectorAll("cssSelector") : css선택자로 복수의 elements를 엑세스한다.

Changing HTML Element(요소의 속성과 내용 수정)

요소 속성 및 내용 수정 테스트

  • element.innerHTML = "<>new html<>" : 요소 내부의 컨텐츠를 변경. 이때 요소(태그)를 추가할 수 있다.

  • element.innerText = "text" : 요소 내부의 텍스트를 변경

  • element.outerHTML = "html" : 요소(엑세스한 element포함) 전체를 변경
  • element.outerText = "Text" : 요소 전체를 text로 변경
  • element.attribute(속성이름) = "value" : 요소의 속성을 수정 or 값을 부여
    [id 이름 검색만 해볼까: ]
  • element.style.property = "new style" : 요소의 스타일을 변경
    [#changingText의 스타일 : ]
    [#changingText의 폰트색 : ]
  • element.setAttribute(속성이름, value) : 요소에 속성과 값을 부여
  • element.removeAttribute(속성이름) : 요소의 속성을 제거
  • element.remove() : 요소를 삭제

Global(전역) 이벤트 속성

HTML요소에 부여된 이벤트 속성으로 브라우저에서 이벤트 작업을 트리거 하는 기능이 있다.
요소 on으로 시작하는 이벤트 속성으로 html dom과 같이 생성되기 때문에 전역 이벤트로 지정된다.
window, mouse, keyboard, form(양식), clipboard, drag, media 이벤트가 있다.
